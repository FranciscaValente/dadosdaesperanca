# Dados da Esperança

Plataforma angolana criada para conectar hospitais, bancos de sangue e doadores. Junte-se a nós e salve vidas em Luanda e restantes províncias.

## 🚀 Publicação

Este site está publicado no GitHub Pages:[(https://franciscavalente.github.io/dadosdaesperanca)](https://franciscavalente.github.io/dadosdaesperanca)

## ✨ Funcionalidades

- **Modo Escuro/Claro**: Alternância de tema para melhor acessibilidade.
- **Responsividade**: Adaptado para dispositivos móveis e desktop.
- **Informações em Tempo Real**: Tipos sanguíneos necessários e hospitais em destaque.

## 🛠️ Tecnologias

- HTML5
- CSS3 (Variáveis CSS, Grid, Flexbox)
- JavaScript (jQuery)
